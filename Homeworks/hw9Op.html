Encryption Methods and Statistical Properties

Substitution Ciphers

Historical Context: Dating back to Julius Caesar's time, substitution ciphers were the primary encryption method until the 20th century.

Mathematical Foundation:

  • Bijective mapping function f: P → C where P is plaintext alphabet and C is ciphertext alphabet
  • Key space size: 26! (approximately 4.03 × 10^26) possible combinations

Statistical Properties:

  • Preserves individual letter frequencies: f(a) occurs as often as a
  • Maintains relative frequency distributions: P(f(a))/P(f(b)) = P(a)/P(b)
  • Vulnerable to frequency analysis attacks due to language patterns
  • Zero entropy increase: H(plaintext) = H(ciphertext)

Cryptanalysis Methods:

  • Frequency Analysis: Using known language statistics
  • Pattern Recognition: Identifying repeated sequences
  • Word Structure Analysis: Common words like "the", "and"

Permutation Ciphers

Mathematical Foundation:

  • Bijective function π: {1,...,n} → {1,...,n} where n is block length
  • Key space size: n! possible permutations

Statistical Properties:

  • Preserves global frequency distribution
  • Disrupts local statistics: bigrams, trigrams
  • Position-dependent entropy: H(pos_i) remains constant
  • Block-size dependent security: larger blocks = more security

Attack Vectors:

  • Known-plaintext attacks using pattern matching
  • Statistical analysis of block boundaries
  • Computational complexity: O(n!) for brute force

Modern Encryption (AES/RSA)

AES Properties:

  • Block cipher with 128/192/256-bit keys
  • Confusion through SubBytes transformation
  • Diffusion through MixColumns and ShiftRows
  • Statistical properties:
    • Uniform output distribution: P(c_i) ≈ 1/256 for all bytes
    • High avalanche effect: changing 1 bit affects ~50% of output bits
    • Maximum entropy: H(ciphertext) ≈ 8 bits per byte

RSA Properties:

  • Based on modular exponentiation: c = m^e mod n
  • Security depends on factoring large numbers
  • Statistical independence between input and output
  • Probabilistic encryption recommended for semantic security

Frequency Analysis & Statistical Tests

English Language Statistics:

  • Single letter frequencies:
    • E (12.7%), T (9.1%), A (8.2%), O (7.5%), I (7.0%)
    • Entropy: ~4.2 bits per character
  • N-gram Analysis:
    • Digrams: TH (3.56%), HE (3.07%), IN (2.43%)
    • Trigrams: THE (3.51%), AND (1.59%), ING (1.15%)

Statistical Tests for Encryption:

  • Entropy Tests:
    • Shannon Entropy: H = -Σ p_i log₂(p_i)
    • Min-entropy: H_∞ = -log₂(max(p_i))
  • Randomness Tests:
    • Chi-square test for uniformity
    • Serial correlation coefficient
    • Runs test for independence
